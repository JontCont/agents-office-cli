How to use Companion GUI

1. Setup your Team (Agents Tab)
Before running any tasks, go to the Agents tab and add your agents (e.g., planner, coder, reviewer). If no agents are configured, the run will fail immediately.

2. Apply Changes (Terminal)
Changes to agents or models are saved to agent-office.yaml. To apply these changes and start a new run, you must restart the .\agent-office.exe gui process in your terminal.

3. Monitor the Run Thread
Watch the agents collaborate in real-time. The Telemetry Dashboard tracks your active token usage, estimated cost, and latency for the current session.

4. Supervisor Interruption
If the agents go off track, click Interrupt Run. The process will pause, allowing you to type guidance into the feedback box. Click Resume Execution to inject your feedback and steer the agents.
